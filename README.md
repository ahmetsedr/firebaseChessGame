# firebase Chess Game

* Projeyi GitHub'dan klonlayın
git clone https://github.com/ahmetsedr/firebaseChessGame.git

* Projeye giriş yapın
cd firebaseChessGame

* Gerekli paketleri yükleyin
npm install

* Projeyi çalıştırın
npm run dev
